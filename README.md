# CBT System

A Computer-Based Testing (CBT) system with a login interface and admin portal.

## Project Structure

```
CBT SYSTEM/
├── index.html           # Main landing page
├── login.html           # Login page
└── cbt-pro-homepage/    # Admin dashboard
    ├── index.html       # Admin dashboard home
    ├── admin-portal.html # Admin portal page
    ├── assets/          # Static assets
    │   └── icons/       # Icon files
    ├── css/
    │   └── styles.css   # Main stylesheet
    └── js/
        └── script.js    # Main JavaScript
```

## Features

- **Login System**: User authentication page
- **Admin Portal**: Administrative dashboard for managing CBT system
- **Responsive Design**: CSS styling for responsive layout
- **Interactive UI**: JavaScript-powered functionality

## Getting Started

1. Open `index.html` in a web browser to view the main landing page
2. Navigate to `login.html` to access the login interface
3. Access admin features through `cbt-pro-homepage/admin-portal.html`

## Files

- `index.html` - Main landing page
- `login.html` - User login page
- `cbt-pro-homepage/index.html` - Admin dashboard home
- `cbt-pro-homepage/admin-portal.html` - Admin portal interface
- `cbt-pro-homepage/css/styles.css` - Central stylesheet
- `cbt-pro-homepage/js/script.js` - Client-side functionality
- `cbt-pro-homepage/assets/icons/` - Icon assets
